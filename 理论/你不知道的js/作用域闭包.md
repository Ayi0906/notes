```
function foo(){
    var a=2;
    function bar(){
        console.log(a);
    }

    bar(); // 2
}

foo();
```
- 基于词法作用域的查找规则，函数`bar()`可以访问外部作用域中的变量a（这是一个RHS引用查询）
- `bar`对`a`的引用的方法是词法作用域的查找规则，而这些规则只是闭包的一部分。

```
function foo(){
    var a=2;
    function bar(){
        clg(a);
    }
    return bar;
}

var baz=foo();
baz(); // 2
```
- `foo`执行后，其返回值（`bar`函数）赋值给变量`baz`并调用`baz()`，实际上只是通过不同的标识符引用调用了内部的函数`bar`
